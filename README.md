# JOGO-DE-XADREZ
jogo de xadrez para o projeto da aula da estâcio
que simula o movimento de uma peça de xadrez com base nas escolhas do usuário.

## Como Usar

1.  **Escolha a Peça**:
    - `1` para Torre
    - `2` para Bispo
    - `3` para Rainha

2.  **Escolha a Direção**:
    - `1` para Direita
    - `2` para Esquerda
    - `3` para Frente
    - `4` para Trás
    - `5` para Diagonal

3.  **Escolha a Quantidade de Movimentos**:
    - Um número de `1` a `9`.

O programa então simulará o movimento e exibirá uma mensagem de resumo com a ação realizada. Se uma entrada inválida for fornecida (por exemplo, uma peça ou movimento que não existe), o programa exibirá uma mensagem de erro e será encerrado de forma segura.
